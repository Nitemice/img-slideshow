A lightweight, single-file web app for viewing local images as a slideshow. Runs entirely in the browser, using vanilla HTML, CSS, and JavaScript, without the need to upload anything or even be online.

## Features

- Load images from local folders, without uploading
- Select sub-folders to include or exclude
- Shuffle order of slideshow
- Adjustable slide duration
- Keyboard and touch controls
- Optional screen wake lock (only available when served via HTTPS)

## Usage

1. Clone this repository, or just download `index.html`.
2. Open the HTML file directly in a browser, or serve it locally,
   e.g. `python -m http.server 8000`.
3. Select folders to load, and you're ready to go

### Slideshow Controls

- Play / pause, previous & next buttons
- Fullscreen button
- Slide duration (seconds)
- Swipe left/right on touch devices for navigation
